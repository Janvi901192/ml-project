# Machine Learning Project

## Aim
To create and manage a GitHub repository for a machine learning project using Version Control System.

## Objectives
1. Create GitHub repository
2. Add README file
3. Upload dataset
4. Demonstrate collaboration

## Dataset
This project uses a sample CSV dataset containing Name, Age, and Salary.

## Tools Used
- Git
- GitHub
- Python

## Usage
Clone the repository and run the code to analyze the dataset.
